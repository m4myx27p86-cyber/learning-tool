## 現在のサイトで残る注意点
- [ ] Writing教材は提出型のため、通常のクイズ設定＋進捗一括画面ではなく、従来のWriting提出画面を維持しています。
- [ ] localStorage保存は端末ごとの記録なので、別端末では進捗が共有されません。
- [ ] 一問ずつ送信は通信が不安定な環境で一部失敗する可能性があります。その場合も端末内履歴は残ります。
- [ ] 画像・演出・グラフを増やしすぎると重くなるため、今回はCSS中心の軽量表示にしています。
- [ ] GitHub Pagesはキャッシュが残ることがあるため、更新直後はURL末尾のバージョンやブラウザ更新が必要です。
