# Stable Repair v2 Checklist - 2026-06-03

## 目的
- パスワード/学習者コード入力がスキップされないようにする。
- 教材画面の表示崩れ・重なりを直す。
- Practice Settings 内で総復習の抽出条件と問題一覧を扱えるように戻す。
- 必要箇所以外は変更しない。

## 変更ファイル
- index.html
- css/game-ui-override.css
- js/game-ui-override.js
- js/auth.js
- split-manifest.json
- static-checks.json

## 変更しない方針
- GAS URL は変更しない。
- CSV 読み込み処理は変更しない。
- Speaking Review の出題・採点処理は変更しない。
- 正解エフェクト本体は変更しない。
- 履歴保存ロジックは変更しない。
- 教材カテゴリ・教材キーは変更しない。

- `index.html` と `js/auth.js` は必ず上書きする。
- ブラウザは `Ctrl + F5` または `Command + Shift + R` で強制更新する。
