# 正解エフェクト改善・機能保持 比較表

| 観点 | 編集前 | 編集後 | 失わないように確認した点 |
|---|---|---|---|
| 1. 正解エフェクト | 正解時にカード表示＋紙吹雪。よいが「学習上何が伸びたか」はやや弱い。 | 連続正解数、Best連続、正解数、正答率、+XP、進捗バー、中央バッジ、光る演出を追加。 | 既存の `processAnswer()` の流れ、スコア加算、履歴保存、自動遷移は維持。 |
| 2. 音声 | 既存コードでは視覚演出が中心。 | 正解は上昇音、誤答は短い確認音。5連続以上は少し達成感のある音階。 | ブラウザで音が鳴らない設定でも学習機能は止まらないよう try/catch 化。 |
| 3. 連続正解 | メニュー上の連続学習日はあるが、問題中の連続正解は見えにくい。 | 問題演習中に `quizCorrectStreak` と `bestCorrectStreak` を追加。 | 新しいクイズ開始・リセット時だけ初期化。既存の学習日数表示には干渉しない。 |
| 4. 間違いの意味づけ | 誤答時は正解表示が中心。 | 誤答時に「なぜ伸びるミスか」「次の行動」を教材タイプ別に表示。 | 間違い履歴への保存、今回のミス解き直し機能は維持。 |
| 5. シンプルなイラスト | アイコン中心。 | 「気づく → 選べる → 使える」「誤答 → 原因 → 次の正解」など、学習プロセスのミニ図解を追加。 | 画像ファイルは追加せず、CSSとHTMLだけで軽量化。 |
| 6. 誤り発見→修正 | 誤り発見後、修正並び替えへ進む。 | Step 1 clear 表示を強化し、「誤り発見」から「修正」への意味を表示。 | 修正問題が並び替えである仕様は維持。 |
| 7. ポラリス3 | スマホで読みやすい本文カード・設問カードあり。 | 今回は正解/誤答フィードバックのみ変更。 | ポラリス3の本文表示・二度押し回答CSS/JSには触れない方針。 |
| 8. 四択二度押し・Enter操作 | 四択は二度押し、Enter操作が使える状態。 | 今回の編集では回答操作ロジックに変更を加えず保持。 | `handleQuizKeyboard` と四択選択処理は保持。 |
| 9. パスワード表示 | 学習者コード・教材パスワードの表示切替あり。 | 今回の編集では未変更。 | `togglePasswordField` と各 checkbox イベントを保持。 |
| 10. ターゲット1900・ライティング理論 | 高校英語にターゲット1900、教師用にライティング理論が存在。 | 今回の編集では教材設定を変更しない。 | `TEST_CONFIG` の教材定義を保持。 |
| 11. 動作の重さ | 紙吹雪演出あり。 | 紙吹雪数は増やしたが短時間で消える。画像を追加せず軽量。reduce-motion にも対応。 | 外部ライブラリや画像ファイルを追加しない。 |
| 12. 今後の提案 | 正解/不正解の意味づけは弱め。 | 次段階では「誤答タイプ別グラフ」「今日の弱点トップ3」「正解音のON/OFF設定」を追加可能。 | 先に現在の安定版を保管してから追加するのが安全。 |

## 変更ファイル

- `script.js`: 正解・誤答フィードバック、連続正解、音声、学習プロセス表示を追加。
- `style.css`: 達成カード、誤答インサイトカード、連続正解バッジ、XP、ミニ図解、中央バッジ、スマホ対応CSSを追加。
- `index.html`: 今回は構造変更なし。既存の画面構成を保持。
